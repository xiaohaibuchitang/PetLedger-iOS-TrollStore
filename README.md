# PetLedger iOS TrollStore

PetLedger is a native SwiftUI bookkeeping app prototype with a calming pet companion scene.

## Build IPA Without a Mac

This repository includes a GitHub Actions workflow that builds a TrollStore-friendly IPA on GitHub's macOS runner.

1. Open the `Actions` tab.
2. Run `Build TrollStore IPA`.
3. Download the `PetLedger-TrollStore-IPA` artifact after the workflow finishes.
4. Install the `.ipa` with TrollStore on a supported iPhone.

## What Is Included

- Apple-style bookkeeping dashboard
- Add-expense sheet
- Healing pet companion room
- Pet bed, plant, and coin jar interactions
- Native SwiftUI animated pet, no external image assets required
